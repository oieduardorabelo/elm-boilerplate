# Elm Boilerplate

Basic structure to create modules or small prototypes im Elm

## Installation

Follow the awesome instructions on [elm-lang.org](http://elm-lang.org/install)

## Usage

```
git clone https://github.com/oieduardorabelo/elm-boilerplate
cd elm-boilerplate
elm-reactor
```
